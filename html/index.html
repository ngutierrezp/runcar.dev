A colorful ASCII car
drives through your terminal.

A tiny HTTP server that streams a 16-row, parallax-scrolling ASCII scene — sun, clouds, mountains, alternating trees, a polished red car, and sand-toned roadside rocks — straight into curl.

~/runcar.dev
$ curl runcar.dev

⚡ This is one of 32 frames streamed at ~11 fps (configurable). Each row is a separate parallax layer scrolling at its own speed.

⚙️ Make it yours

Tweak playback on a per-request basis with HTTP headers.

⏩ Speed milliseconds between frames
curl -H "X-Frame-Rate: 60" runcar.dev

Override the default 90 ms interval. The server clamps to a minimum of 30 ms so it stays kind to your CPU.

🎚️ Monochrome strip ANSI escape codes
curl -H "X-No-Color: 1" runcar.dev

Get the plain ASCII version — perfect for terminals that don't speak ANSI, or for embedding in logs.
